**ALL Classes**  
`fade` - (on/off) use class AA Fade   
`fade` - [1,99]% to use AA fade when on   
`spiregroup` - [0,3] set spire to use while grouped and burning. 0=disable  
`spireraid` - [0,3] set spire to use while in a raid and burning. 0=disable
